# مولد IBAN سعودي للتست

أداة ديسكتوب مبنية بلغة Python وواجهة Tkinter، تتيح لك توليد أرقام IBAN تجريبية للبنوك السعودية والتحقق من صحتها.
تم احتساب **Check Digits بشكل صحيح** باستخدام خوارزمية IBAN الرسمية (Mod 97) لتكون أقرب للحقيقة.

> **تنبيه:** الأرقام المولدة للاختبار فقط وليست صالحة لتحويلات مالية حقيقية.

---

## المميزات

- توليد IBAN مطابق للتنسيق الرسمي السعودي (24 خانة)
- التحقق من صحة أي IBAN سعودي مع كشف اسم البنك تلقائيا
- عرض IBAN بتنسيق مقروء (مجموعات من 4 خانات)
- سجل آخر 10 أرقام IBAN تم توليدها
- نسخ سريع مع رسالة تأكيد
- وضع داكن / فاتح
- اختصارات لوحة المفاتيح (Enter للتوليد، Ctrl+C للنسخ)

## التثبيت والتشغيل

### 1. المتطلبات

- Python 3.7 أو أحدث
- مكتبة `pyperclip`
- مكتبة `tkinter` (مضمنة مع Python)

### 2. تثبيت المتطلبات

```bash
pip install pyperclip
```

### 3. تشغيل البرنامج

```bash
python iban.py
```

## طريقة الاستخدام

### توليد IBAN

1. اختر البنك من القائمة المنسدلة
2. اضغط **توليد IBAN** أو اضغط **Enter**
3. سيظهر الرقم بتنسيق مقروء مثل: `SA65 8000 0112 6080 1029 7476`
4. اضغط **نسخ** أو **Ctrl+C** للنسخ إلى الحافظة

### التحقق من IBAN

1. الصق أو اكتب رقم IBAN في حقل التحقق
2. اضغط **تحقق** أو **Enter**
3. سيظهر اسم البنك ونتيجة التحقق (صحيح / غير صحيح)

### اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| Enter | توليد IBAN جديد |
| Ctrl+C | نسخ IBAN إلى الحافظة |
